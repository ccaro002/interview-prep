# interview-prep

Daily algorithm and data-structure practice in Java, organized by pattern
(arrays & hashing, two pointers, sliding window, trees, graphs, dynamic
programming, and so on). Each solution includes a short write-up of the
approach, time/space complexity, and what I'd do differently. Part of a
structured return to backend engineering after six years as a Software
Development Engineer at Amazon (Retail, Amazon Go, AWS).
